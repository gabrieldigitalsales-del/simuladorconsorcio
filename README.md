# Vibra Soluções — Simulações de Consórcio

Projeto Vite + React reconstruído conforme o layout aprovado.

## Recursos

- 4 cenários editáveis diretamente no cabeçalho da planilha
- cores automáticas e diferentes para cada cenário
- valores inicialmente vazios, com placeholders
- nomes das informações fixos
- seletor em cada informação para incluir ou remover do PDF
- seletor em cada cenário para incluir ou remover do PDF
- pré-visualização do PDF na própria tela
- exportação por impressão em A4 paisagem
- sem linha de cliente, vendedor ou validade
- salvamento local no navegador
- responsivo; rolagem horizontal restrita à tabela
- fonte padrão do sistema (`system-ui`)

## Rodar localmente

```bash
npm install
npm run dev
```

## Gerar versão de produção

```bash
npm run build
```

A pasta `dist` já acompanha o projeto pronta para publicação.
